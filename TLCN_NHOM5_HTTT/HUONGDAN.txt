OS: Windown 10/11
Python: 3.10.4(Đừng dùng Python 3.12 vì nó lỗi)

MỞ Command Prompt TẠI THƯ MỤC NÀY VÀ THỰC HIỆN CÁC BƯỚC SAU

BƯỚC 1: Tải các thư viện để chạy file

-- Tại vị trí thư mục code gõ câu lệnh sau vào Command Prompt

pip install -r requirements.txt

BƯỚC 2: Dùng streamlit để chạy file Home.py trong local

-- Tại vị trí thư mục code gõ câu lệnh sau vào Command Prompt

streamlit run Home.py    

-- Nếu streamlit bắt nhập email thì để trống

-- streamlit sẽ chạy ở địa chỉ: http://localhost:8501
